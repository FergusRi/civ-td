# Civ TD 🏰

A **WorldBox × RimWorld** civilisation tower defence game. Build a living colony, survive increasingly brutal waves of enemy factions, and last as long as you can.

## Gameplay
- **Plan phase**: Build production buildings, walls, towers, and houses
- **Wave phase**: Defend against faction attacks — each faction has weaknesses to specific tower types
- Last as many waves as possible

## Factions
| Faction | Attacks from | Weak to |
|---|---|---|
| 🐗 Goblin Horde | North | Archer Tower, Cannon |
| 🪓 Orc Warband | West | Ballista, Cannon |
| 💀 Undead Legion | South | Mage Tower |
| 🏹 Dark Elf Raiders | East | Archer Tower, Lightning Rod |
| 🗿 Stone Giants | Northwest | Ballista, Catapult |
| 🌑 Shadow Cult | Any edge (wave 15+) | Mage Tower |

## Running
Open `index.html` in a browser (no build step required — pure ES modules).

## Controls
- **WASD / Arrow keys**: Pan camera
- **Scroll wheel**: Zoom
- **Click build panel**: Select building type
- **Click map**: Place building
- **Escape**: Cancel placement
- **Start Wave button**: Begin the wave
